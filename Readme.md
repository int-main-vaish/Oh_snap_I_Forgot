# 🧠 Oh Snap, I Forgot!

An AI-powered productivity assistant that converts a messy task dump into a prioritized action plan using Google Gemini AI.

## ✨ Features

- 🤖 AI-powered task prioritization
- 📊 Smart urgency analysis
- 💭 "What If?" consequence simulator
- 📅 Google Calendar integration
- ⏳ Countdown timers
- 📈 Panic Meter
- 🐶 Interactive motivational pet
- 📱 Responsive UI

## 🛠️ Tech Stack

- Python
- Flask
- Google Gemini API
- HTML
- CSS
- JavaScript
- Docker
- Google Cloud Run

## 🚀 Installation

```bash
git clone https://github.com/int-main-vaish/Oh_snap_I_Forgot.git
cd Oh_snap_I_Forgot

pip install -r requirements.txt

# Create a .env file
GEMINI_API_KEY=your_api_key
```

Run the application:

```bash
python main.py
```

Visit:

```
http://127.0.0.1:8080/```
